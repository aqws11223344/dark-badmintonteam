# 黑武士 Bot 指令速查

> 最後更新：2026-04-18

## 📢 公開指令（所有隊員可用）

### `/help` 或 `/說明`
顯示公開指令列表。

### `/登記` 或 `/add`
取得成績登記連結（LIFF 表單）。隊員自己從下拉選賽事。

```
你：/登記
Bot：🏸 成績登記
     👉 https://liff.line.me/2009823919-BC2lq8hx
     從下拉選擇賽事後填寫
```

### `/賽事列表`
顯示目前有哪些賽事在下拉選單裡。

```
你：/賽事列表
Bot：📋 目前賽事列表：
     2026清晨盃
     2026會長盃
```

### `/<賽事名稱>` — 查詢某場比賽的全部得獎紀錄

```
你：/2026清晨盃
Bot：🏆 2026清晨盃 成績紀錄（3 筆）
     ───────────
     • 香菇｜混雙 冠軍（公開組） / 搭檔:小美
     • 阿強｜男雙 亞軍（甲組） / 搭檔:李振偉
     • 大頭｜男單 四強（公開組）
```

> 注意：只有「已註冊的賽事名稱」才會回應。打 `/亂打一通` 會**靜默**（不回）。

### `/我的ID` 或 `/myid`
顯示自己的 LINE User ID（想申請當管理員時用）。

```
你：/我的ID
Bot：你的 LINE User ID：
     U1234abcd...
```

## 🔐 管理員指令（非管理員打會顯示警告或靜默）

### `/dhelp`
顯示管理員專屬指令列表。**非管理員打會被 bot 當沒看見**（不洩漏指令存在）。

### `/開單 <賽事名稱>`
發起某場比賽的成績登記。會自動把賽事加入下拉，並在群組貼出 LIFF 連結。

```
你：/開單 2026清晨盃
Bot：📋 2026清晨盃 成績登記
     👉 https://liff.line.me/...
     （每人可填多項，可隨時回來修改）
```

### `/新增賽事 <名稱>`
只加到下拉選單，不發連結。

```
你：/新增賽事 2026會長盃
Bot：✅ 已新增賽事：
     2026會長盃
```

### `/刪除賽事 <名稱>`
從下拉選單移除。已填的舊資料不會動。

```
你：/刪除賽事 2026會長盃
Bot：🗑 已刪除賽事：
     2026會長盃
```

### `/新增管理員 <UserID>`
把某人加入管理員。

**流程：**
1. 對方加 bot 好友
2. 對方傳 `/我的ID` 拿到 ID
3. 對方把 ID 傳給你
4. 你打 `/新增管理員 U1234...`

```
你：/新增管理員 U1a2b3c4d5e6f...
Bot：✅ 已新增管理員：
     U1a2b3c4d5e6f...
```

### `/刪除管理員 <UserID>`
移除管理員權限。

### `/管理員列表`
顯示目前所有管理員。

```
你：/管理員列表
Bot：👮 管理員列表：
     • 蔡秉諺（U1234...）
     • 其他人（U5678...）
```

## 🔑 隱藏指令（不列在任何 help）

### `/addme`
**第一位管理員自助 bootstrap 專用**。

- **Store 的 admins 是空的 → 把你加成管理員**
- **已有管理員 → 靜默不回**（避免被濫用）

```
你：/addme       （第一次）
Bot：✅ 你已成為管理員：蔡秉諺
     輸入 /dhelp 查看管理員指令

你：/addme       （第二次或他人打）
Bot：（靜默）
```

> ⚠️ 使用前提：**你必須先加 bot 為好友**，否則 bot 拿不到你的 userID。
>
> 如果不小心把自己的管理員刪了，可以去 Google Sheet 的 `admins` 分頁手動清空，再打一次 `/addme` 重新 bootstrap。

## 🎯 典型使用情境

### 一：剛部署完，設定第一位管理員
```
你 → bot：加好友
你 → bot：/addme
Bot → 你：✅ 你已成為管理員：蔡秉諺
```

### 二：週末比完清晨盃，開始蒐集成績
```
你（管理員）→ 群組：/開單 2026清晨盃
Bot → 群組：📋 2026清晨盃 成績登記 👉 [LIFF 連結]

隊員 A → 點連結 → 填「香菇 混雙 冠軍」→ 送出
隊員 A → 再新增一筆 → 填「香菇 男雙 亞軍」→ 送出
隊員 B → 點連結 → 填「阿強 男單 四強」→ 送出
...
```

### 三：查詢成績
```
任何人 → 群組：/2026清晨盃
Bot → 群組：🏆 2026清晨盃 成績紀錄（5 筆）...
```

### 四：隊員遺失連結
```
隊員 → 群組：/登記
Bot → 群組：🏸 成績登記 👉 [LIFF 連結]
```

### 五：加新管理員
```
新人 → bot：加好友
新人 → bot：/我的ID
Bot → 新人：你的 LINE User ID：U9999...
新人 → 你：把 ID 告訴你
你 → bot：/新增管理員 U9999...
Bot → 你：✅ 已新增管理員
```

## 🗃️ Google Sheet 分頁說明

部署後 bot 會自動在你的 Sheet 建立：

| 分頁 | 欄位 | 用途 |
|------|-----|------|
| `2026`（當年度）| 時間 / 輸入人 / 姓名 / 賽事 / 組別 / 項目 / 搭檔 / 名次 / 備註 | 成績資料 |
| `tournaments` | 賽事 | 賽事下拉清單 |
| `admins` | UserID / Name / AddedAt | 管理員清單 |

> 緊急情況（例如把自己管理員刪掉）可以直接去 Sheet 手動修改。
